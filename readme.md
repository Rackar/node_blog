# 安装运行

如相应工具未安装参考教程文章中的[基础开发环境搭建](https://www.jianshu.com/p/d14ba8251237)及[2 分钟搭建 MongoDB](https://www.jianshu.com/p/d5713c8d4df3)

命令行下执行：

```
git clone https://github.com/Rackar/node_blog
cd node_blog
npm i
npm start
```

(vs code 调试无需 npm start,直接 F5。)
